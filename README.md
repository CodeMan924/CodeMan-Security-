# CodeMan Security-
CodeMan's Security is a Security App which does basic things like scanning your device for malware. This program is open source which means it's completely free. 
The program checks the disk for any malware or vulnerabilities and the program tells the error to you, you can choice to fix it or simply close it. 

Though you cannot open it directly. You have to install dependencies to get the app. Please note this app is only compatible with windows 11 or maybe 10, here are the instructions:
first go to your command prompt (press the windows logo and click R, this opens the run, enter cmd) then type this command : pip install customtkinter psutil
This installs the essentials to run the program.
Other steps:
Install the program, Click on code and press Download ZIP. 
Once installed extract the file
Run the program CodeMan Security.exe
